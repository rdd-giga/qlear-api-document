## 类型说明

文档中给出了全部返回值的类型并对部分字段进行了注解

类型书写采用的typescript的格式。

| 符号      | 含义                         |
| -------- | ---------------------------- |
| ?        | 值可能为 null 或 字段不存在     |
| &#124;   | 或                           |
| []       | 数组且数组元素均为该类型         |
| ""       | 特定的字符串                   |
